# CONTEXT

## 이 모듈이 하는 일
- DB 스키마를 기준으로 생성된 타입 정의를 보관한다.

## 파일 구조와 역할
- `types/supabase.ts`: Supabase/DB 스키마 기반 타입 아웃풋
- `types/mdx.d.ts`: MDX 모듈 import 타입 선언

## 핵심 설계 결정
- 수동 편집보다 생성 커맨드(`bun run typegen`)를 우선한다.

## 사용 패턴
- 서비스/쿼리 코드에서 타입 힌트가 필요할 때 참조한다.
- 마이그레이션 변경 직후 타입을 재생성해 문서/코드와 일관성을 유지한다.

## 확장 시 고려사항
- 생성 결과를 커밋하기 전 불필요한 잡음(diff)을 검토한다.
